# 📱 Neogranada Conecta

Aplicación móvil desarrollada con Ionic y Angular para la gestión de préstamos académicos en la Universidad Militar Nueva Granada.

---

## 🚀 Descripción

Neogranada Conecta es una aplicación móvil diseñada para optimizar el proceso de préstamo y reserva de recursos universitarios, permitiendo a estudiantes y administradores gestionar solicitudes de manera rápida, organizada y centralizada.

La plataforma permite:

- Reservar aulas
- Solicitar laboratorios
- Gestionar material lúdico
- Consultar disponibilidad
- Administrar préstamos
- Visualizar calendarios
- Recibir notificaciones
- Gestionar solicitudes en tiempo real

## ⚙️ Instalación

### Clonar el repositorio

```bash
git clone https://github.com/TU-USUARIO/neogranada-conecta.git
```

### Entrar al proyecto

```bash
cd neogranada-conecta
```

### Instalar dependencias

```bash
npm install
```

### Ejecutar en navegador

```bash
ionic serve
```

---

## 📱 Generar APK Android

```bash
ionic build
npx cap sync android
npx cap open android
```
